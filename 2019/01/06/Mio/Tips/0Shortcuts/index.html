Eclipse

Created Friday 21 December 2018

Toogle Wrap alt+ shift + y

*Cambiar Entre tabs: Ctrl + E o tambien: Ctrl + PgUp/PgDn

Collapse All Ctrl + / (Numpad)
Uncollapse All Ctrl + * (Numpad)
Collapse Ctrl + - (Numpad)
Uncollapse Ctrl + + (Numpad)

Ingresar a funcion: F3 or Ctrl + Click
Volver/Ir a funcion anterior: Ctrl + Right/Left

Vscode

Created Wednesday 28 November 2018

Zen mode Ctrl+k+z

go back from definition: BACK ctrl + alt + -
Go to the definition F12

Auto wrap… (para ventanas pequenhias ) ctrl +z alt+z

Collapse functions (Fold) Ctrl + shfit + [ Fold all ctrl + k ctrl +0
Expand functions (Unfold) Ctrl + shfit + ] Unfold all ctrl+k ctrl+J

Deindent Shift + tab

Select multiple lines ctrl + Shift + alt + UP/DOWN o tb Alt + click (one by one)
Change a word (in the document) 1. ctrl +f (to select all the same words) 2. alt + enter (to edit)
